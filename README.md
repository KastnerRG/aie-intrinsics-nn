# Optimized Intrinsics based AIE Programming

In waiter, do:
```bash
source /tools/Xilinx/Vivado/2024.1/settings64.sh && source /opt/xilinx/xrt/setup.sh
```

To run simulation, enter a directory and 
```bash
make clean
make analyze
```
